==一、Git下载安装==

  1. git官网下载git安装包

​ 点击Binary installer下的 installer选择安装包下载

​ 在MAC 客户端 输入 git version 查看git是否安装成功

  1. 在gitee官网注册账户并登陆

==二、配置公钥==

如果不配置公钥，每次上传文件到远程仓库都需要用户名和密码验证。

点击查看SSH公钥配置

  1. 复制一下命令，去git终端执行生成 SSH Key：

注：”Gitee SSH Key”可以任意命名

输出，如：

  • ==中间通过三次回车键确定==
  1. 查看生成的 SSH 公钥和私钥：

进入上面地址文件夹，找到文件：

  • 私钥文件 id_ed25519
  • 公钥文件 id_ed25519.pub
  1. ==用记事本打开公钥文件== id_ed25519.pub，复制公钥

​ 将公钥粘贴到 gitee 后台公钥文本框内，标题命名，如：Python Study

==确认公钥是否成功添加==，

​ 在终端输入

​ ssh -T git@gitee.com

​ 然后输入：yes

​ 显示以下内容，则成功

​ Hi USERNAME! You've successfully authenticated, but GITEE.COM does not provide shell access.

==四、下载远程仓库到电脑==

​ Https 和 ssh 都可以，这里我们选择 ssh ,复制地址

​ ==在终端执行下载命令==

​ git clone git@gitee.com:elevenworld/wef.git

​ 在电脑文件夹 /Users/xiejincheng 可看到下载的远程仓库文件夹Python Study

==五、文件上传至远程仓库==

​ 在Python Study放入需要上传的文件

​ 当前终端地址为桌面，在当前终端输入

​ cd Python Study

​ ==将终端地址切换至Python Study文件夹==

​ ==分别输入以下三个命令==

  • git add . 把所有内容放进暂存区
  • git commit -m "写一句描述该文件夹的信息"
  • git push origin master

==六、从仓库下载到电脑==

进入远程仓库，复制https地址

https://gitee.com/elevenworld/phython-study.git

将该地址输入终端 即可执行下载远程仓库内文件

在终端输入以下信息：

git clone https://gitee.com/elevenworld/phython-study.git
